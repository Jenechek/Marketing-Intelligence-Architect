# Marketing Intelligence

Локальное веб-приложение для мониторинга сайтов, конкурентов, структуры сайта, поисковой выдачи и возможных причин изменения трафика.

## Статус

Проект находится на этапе подготовки.

Рабочие функции ещё не реализованы.

## Основные цели

- мониторинг изменений на сайтах конкурентов;
- карта структуры сайта;
- история изменений поисковой выдачи;
- анализ возможных причин падения трафика;
- локальная работа без обязательных платных сервисов.

## Основные ограничения

- первая версия запускается на компьютере пользователя;
- основные функции не зависят от платного хостинга;
- обязательные платные API не используются;
- продукт ориентирован на одного локального пользователя;
- интерфейс создаётся на русском языке.

## Планируемый стек
