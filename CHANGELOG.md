# CHANGELOG.md

Все заметные изменения проекта фиксируются в этом файле.

Формат основан на принципе: что изменилось для пользователя, а не только какие строки кода были изменены.

## [Unreleased]

### Добавлено

- минимальное FastAPI-приложение с русскоязычной стартовой страницей;
- автоматическая подготовка локальных папок `data`, `logs` и базы SQLite через SQLModel;
- базовое журналирование в локальный файл и консоль;
- сквозной тест запуска, HTTP-ответа и подготовки SQLite;
- зависимости Python и подробная инструкция запуска на Windows 10 и Windows 11;
- `.gitignore` для локальных данных, секретов, окружений и временных файлов;
- согласованный `REQUIREMENTS.md` с бизнес-, функциональными и нефункциональными требованиями;
- открытые вопросы и критерии приёмки по этапам разработки;
- утверждённые Markdown-документы в корне GitHub-репозитория;

- запрет на переход к следующей задаче до завершения текущей.

- обязательное обновление документов после каждой успешно завершённой задачи;

- журнал выполнения `PROJECT_LOG.md`;

- паспорт продукта;
- дорожная карта;
- журнал архитектурных решений;
- файл текущих задач;
- инструкции для Codex;
- UX/UI-принципы;
- начальный README.

### Изменено

- выбран SQLModel для работы с SQLite;
- уточнены правила безопасного обхода, анализа структуры, импорта данных и проверки гипотез;
- документация Google Drive и GitHub приведена к единому согласованному состоянию;
- зафиксирован local-first подход;
- зафиксировано требование отсутствия обязательных платных зависимостей;
- зафиксирован принцип прогрессивного раскрытия функций.

### Исправлено

- в инструкции для Windows добавлен способ запуска `pytest` с отдельным временным каталогом при ошибке `PermissionError`.
