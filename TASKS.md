## Текущая задача

**Идентификатор задачи:** TASK-0002 — подготовлена к проверке 2026-07-14

Создать экран списка сайтов и форму добавления сайта.

## Ожидаемый результат

- система показывает пустой список и ранее добавленные сайты;
- пользователь может добавить сайт, указав название и полный URL;
- некорректный URL не сохраняется, а форма показывает понятную ошибку;
- данные сайта сохраняются в SQLite и доступны после перезапуска.

- [x] Добавить серверную проверку URL и понятные ошибки.
- [x] Сохранять сайт в SQLite одной транзакцией.
- [x] Проверить сохранение после перезапуска приложения.
- [x] Добавить автоматические тесты основного и ошибочного сценариев.
- [x] Обновить затронутую документацию.

## Критерии завершения

- список сайтов доступен через основной экран;
- корректный сайт добавляется через интерфейс и остаётся после перезапуска;
- некорректный URL не попадает в базу;
- ошибки формы понятны и доступны для вспомогательных технологий;
- автоматические и фактические проверки проходят;
- нет новых платных или внешних зависимостей.
