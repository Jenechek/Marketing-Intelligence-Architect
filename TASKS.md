## Текущая задача

**Идентификатор задачи:** TASK-0003 — согласована 2026-07-14

Добавить редактирование названия и URL существующего сайта.

## Ожидаемый результат

- у каждого сайта доступно вторичное действие «Редактировать»;
- отдельная страница редактирования показывает текущие название и URL;
- пользователь может сохранить корректные изменения или отменить редактирование;
- некорректные данные не сохраняются, а форма показывает понятную ошибку;
- сохранённые изменения доступны после перезапуска приложения;
- неизвестный идентификатор сайта обрабатывается понятным русскоязычным сообщением.

## Подзадачи

- [ ] Добавить получение сайта по идентификатору.
- [ ] Добавить страницу редактирования с предварительно заполненными полями.
- [ ] Добавить вторичное действие «Редактировать» в строке сайта.
- [ ] Повторно использовать правила проверки названия и URL.
- [ ] Сохранять изменения в SQLite одной транзакцией.
- [ ] Не изменять данные при ошибке проверки или отмене.
- [ ] Обработать неизвестный идентификатор сайта.
- [ ] Добавить автоматические тесты успешного и ошибочных сценариев.
- [ ] Проверить сохранение после перезапуска приложения.
- [ ] Обновить затронутую документацию.

## Критерии завершения

- название и URL сайта редактируются через интерфейс;
- страница содержит одно визуально приоритетное действие «Сохранить изменения»;
- действие «Отмена» безопасно возвращает к списку без изменений;
- некорректный URL не попадает в базу, а прежние данные сохраняются;
- изменения остаются доступными после перезапуска;
- неизвестный сайт не вызывает необработанную ошибку;
- удаление сайта и изменение правил дубликатов URL не входят в задачу;
- автоматические и фактические проверки проходят;
- нет новых платных или внешних зависимостей.
